# Fachschaft Informatik Tübingen e.V.

Dieses Repository enthält alle wichtigen Daten und Projekte zum Verein _Fachschaft Informatik Tübingen e.V._,
dem Förderverein der Fachschaft Informatik.

Da das für uns geltende Vereinsrecht auf deutscher Sprache verfasst ist und viele Fachbegriffe sich nur unschön übersetzen lassen,
wird dieses Repository ausnahsmweise auf Deutsch geführt. Bitte dennoch englische Commit-Messages verwenden. 

# Inhalt

### Unterordner `satzung`
Die Satzung unseres eingetragenen Vereins, wie sie auch beim Vereinsregister eingereicht wurde.

### Unterordner `flyer`
Ein Entwurf für einen Werbeflyer, der den Förderverein grob erklärt und die Möglichkeiten der Mitgliedschaft aufzeigt.

# Geschichte
Der Verein _Fachschaft Informatik Tübingen e.V._ wurde am 27. November 2025 gegründet von: Alexander Goetz, Kai Priester,
Larissa Reichart, Nico Faden, Samuel Füßinger, Linus Szillat und Sebastian Walter.
